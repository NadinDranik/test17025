# ТЕХНИЧЕСКОЕ ЗАДАНИЕ

## Проект

Создать закрытое веб-сообщество для специалистов аккредитованных лабораторий под названием:

**«Гид PRO ГОСТ ISO/IEC 17025-2019»**

Проект должен представлять собой современный адаптивный сайт с регистрацией пользователей, бесплатным общим чатом, закрытым PRO-разделом по подписке и административной панелью.

На данном этапе не требуется интернет-магазин, корзина, автоматические платежи и сложная CRM.

Основная задача проекта — создать профессиональное экспертное пространство для общения специалистов лабораторий.

---

# PRO-РАЗДЕЛ

Создать закрытый раздел.

Доступ только пользователям с активной подпиской.

ВАЖНО:

Темы НЕ создавать автоматически.

Все темы создаются администратором вручную через админ-панель.

Система должна позволять администратору самостоятельно:

— создавать тему

— редактировать тему

— удалять тему

— закреплять тему

— скрывать тему

Пользователи видят только созданные администратором темы.

# УПРАВЛЕНИЕ ПОДПИСКАМИ

На данном этапе НЕ использовать автоматические платежные системы.

Оплата осуществляется напрямую через администратора.

Администратор должен иметь возможность:

— вручную выдать PRO-доступ

— вручную продлить PRO-доступ

— вручную отключить PRO-доступ

— установить дату окончания подписки

— изменить дату окончания подписки

— видеть список активных подписчиков

— видеть список пользователей с истекшей подпиской

— видеть список пользователей без подписки

— быстро продлевать подписку одним нажатием

Система должна автоматически:

— предупреждать администратора о скором окончании подписки

— уведомлять пользователя о скором окончании подписки

— блокировать доступ к PRO-разделу после окончания подписки

При этом аккаунт пользователя сохраняется.
